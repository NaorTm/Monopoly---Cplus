# Monopoly (C++)

Console-based Monopoly implementation with a game engine, player management, and board logic.

## Features
- Game engine with turn flow and dice rolls
- Board and slot definitions
- Player assets and decision handling
- Uses `board.txt` for board setup

## Build and Run
Open `boardGame.sln` in Visual Studio and build.

Or compile in the `boardGame/` folder with your C++ compiler.
